##General Project Information

This project is an online store website for clothing and yoga accessories. It was created to fulfill the requirements of the course "Programming Frameworks in Web Applications."
The project was developed using the following technologies:
- React JS: A JavaScript library for building user interfaces.
- Express & Node JS: Used to create the application's backend, handle HTTP requests, and manage business logic.
- MongoDB: A NoSQL database used for storing data about products and users.

The following tools were used:
- CSS: For managing user interface styles.
- Multer: A Node.js middleware for handling file uploads (e.g., images) to the server.
- bcrypt: A library for password encryption.
- JWT: A standard for authorization, used to securely authenticate users in the application.
- Cors: Allows controlled access to resources on different domains in a secure way.
- React Router: A library for managing routing in React applications.

##Installation

1. Download and unzip the 'Projekt' folder.
2. In the project folder, open an integrated terminal for the 'frontend' folder, and then run the following commands:
    2.1. npm install
    2.2. npm install react-router-dom
3. In the project folder, open an integrated terminal for the 'backend' folder, and then run the following commands:
    3.1. npm install express
    3.2. npm install jsonwebtoken
    3.3. npm install mongoose
    3.4. npm install multer
    3.5. npm install cors
4. In the project folder, open an integrated terminal for the 'admin' folder, and then run the following commands:
    4.1. npm install
5. In the terminal of the 'frontend' folder, run the command: npm start.
6. In the terminal of the 'backend' folder, run the command: node .\index.js.
7. In the terminal of the 'admin' folder, run the command: npm run dev.
8. The store website will be available at: http://localhost:3000/.
9. The admin panel will be available at: http://localhost:5173/.
